== Description ==

Man kann nun, ganz einfach seinen Fahrzeugbestand von Mobile.de mit einem Framework auf seiner Webseite anzeigen lassen.

Webseite: https://domisiding.de

== Installation ==

- Sie benötigen ein Mobile.de Haendlerkonto

- Wenn Sie auf ihrem Haendlerkonto eingeloggt sind, befindet sich im Administration Bereich eine CustomerId, mit dieser ID koennen Sie dieses Framework betreiben.

- Wenn Sie ihre CustomerID gefunden haben, gehen Sie auf die Einstellungen von Framework Mobile.

- Dort tragen Sie die CustomerID wie im Link Beispiel ein: https://home.mobile.de/home/index.html?partnerHead=false&colorTheme=default&customerId=[IHRE CUSTOMERID] (ohne die [])
